# GlucoCoach

A personal diabetes management platform built to make CGM data actually understandable. Connects to [Nightscout](https://nightscout.github.io/) to pull continuous glucose monitor readings and uses AI to generate plain-English summaries of glucose patterns.

Built for myself first — because the raw data from a CGM is overwhelming without the right tools to interpret it.

## api/

Python backend that connects to Nightscout, computes clinical glucose metrics, and generates AI-powered insights using Claude.

**Key features:**

- Nightscout integration — fetches real-time and historical CGM readings
- Time in Range (TIR), GMI, glucose variability (CV / Std Dev)
- Ambulatory Glucose Profile (AGP) — 24h glucose patterns by percentile
- Dawn phenomenon detection
- Smart bolus timing recommendations based on current glucose + glycemic index
- Meal logging with glucose correlation — tracks which meal types cause the biggest spikes
- Hypo event tracking
- AI daily summaries — sends aggregated metrics to Claude, returns plain-English insight
- PDF report generation

**Stack:** FastAPI · PostgreSQL · SQLAlchemy 2.0 · Pydantic v2 · Pandas · NumPy · Docker

**Run locally:**

```bash
cd api
cp .env.example .env   # add your Nightscout URL + secret
uv sync
uv run fastapi dev
# API: http://localhost:8000
# Docs: http://localhost:8000/docs
```

**Run with Docker:**

```bash
cd api
docker compose up -d
```

## Clinical metrics implemented

| Metric              | Description                                    | Target |
| ------------------- | ---------------------------------------------- | ------ |
| **TIR**             | Time In Range (70–180 mg/dL)                   | > 70%  |
| **GMI**             | Glucose Management Indicator — estimated A1c   | —      |
| **CV**              | Coefficient of Variation — glucose variability | ≤ 36%  |
| **Std Dev**         | Standard deviation of glucose readings         | —      |
| **Dawn Phenomenon** | Morning glucose rise (2 AM → 7 AM delta)       | —      |

See [`api/CALCULATIONS.md`](./api/CALCULATIONS.md) for the full clinical formulas.

## Environment variables (api)

| Variable            | Description                       |
| ------------------- | --------------------------------- |
| `NIGHTSCOUT_URL`    | Your Nightscout instance URL      |
| `NIGHTSCOUT_SECRET` | Your hashed Nightscout API secret |
| `DATABASE_URL`      | PostgreSQL connection string      |
| `ANTHROPIC_API_KEY` | For AI insight generation         |

Copy `api/.env.example` to `api/.env` and fill in your values.
